# CV Portfolio Website

Website CV/Portfolio sederhana yang dibuat untuk keperluan tugas.

## Fitur
- Desain modern dengan tema gelap
- Animasi smooth dan efek hover
- Responsive untuk semua ukuran layar
- Navigasi yang mudah digunakan
- Portfolio showcase

## Cara Menggunakan

1. **Edit Konten**
   - Buka `index.html` dan ganti placeholder text dengan informasi Anda
   - Update nama, deskripsi, skills, dan proyek portfolio
   - Ganti informasi kontak (email, telepon, lokasi)

2. **Ganti Gambar**
   - Untuk foto profil, tambahkan gambar ke file dan update di CSS `.profile-img`
   - Untuk portfolio items, ganti gradient dengan `background-image: url('path/to/image.jpg')`

3. **Deploy ke GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin <URL-REPOSITORY-ANDA>
   git push -u origin main
   ```

4. **Aktifkan GitHub Pages**
   - Buka Settings repository > Pages
   - Pilih branch `main` dan folder `root`
   - Website akan aktif di `https://username.github.io/repository-name`

## Teknologi
- HTML5
- CSS3 (dengan custom properties dan animasi)
- Vanilla JavaScript
- Google Fonts (Poppins & Inter)

## Lisensi
Bebas digunakan untuk keperluan tugas dan pembelajaran.
